## Install

```bash
git clone https://github.com/orihalcum/shipper-id-dashboard.git
npm install 
```
## Run Project

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
```bash
npm start
```

## Running the tests

Launches the test runner
```bash
npm run test
```
